Significado de los colores

ROJO: El color rojo puede significar los siguientes mensajes.
- Meta Individual: Te figurara este color si no has cumplido uno de tus 2 metas ya sea viajes o asociados.
- Meta Grupal: Te figura este color si las metas grupales de tus niveles subordinados no están siendo cumplidas. El color rojo generado por el incumplimiento de una meta grupal puede cambiar a amarrillo, siempre y cuando haya establecido una comunicación para solucionar este problema.

AMARILLO: Este color es para hacer referencia a que ya entablé una comunicación con mis niveles subordinados.

VERDE: Significa que has cumplido o superado tus metas individuales, al igual que tus socios subordinados cumplieron o superaron las suyas.

NEGRO: Significa que este asociado se encuentra suspendido.
